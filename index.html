押して、話すだけ。

ホットキーを押して話すと、文字起こしされたテキストがカーソル位置にそのまま入力されます。

アカウント登録も API キーも不要。インストールしたらすぐに音声入力できます。

Mac のインストール

  1. ダウンロードした DMG を開き、voicekey.app を「Applications」フォルダにドラッグします。
  2. 初回起動時に「開発元を検証できません」等の警告が出たら、お使いの macOS に応じて開いてください。

macOS 14 (Sonoma) の場合

アプリケーションフォルダで voicekey.app を右クリック →「開く」→「開く」

macOS 15 (Sequoia) 以降の場合

一度ダブルクリックして警告を閉じる → システム設定 → プライバシーとセキュリティ → 下の方にある「このまま開く」

この警告について: 配布方式によるもので、アプリ自体の安全性に問題はありません。初回に一度開けば、以後は通常どおりダブルクリックで起動できます。

初期設定

初回起動時にマイク・入力監視・アクセシビリティの許可を求められます。すべて許可してください（音声入力と自動貼り付けに必要です）。

Windows のインストール

  1. ダウンロードした voicekey-setup-X.Y.Z.exe をダブルクリックします。
  2. 「WindowsによってPCが保護されました」（SmartScreen）が出たら、「詳細情報」→「実行」の順にクリックしてください。
  3. あとはインストーラの指示に従って進めるだけです（管理者権限は不要です）。
この警告について: 配布方式によるもので、アプリの安全性に問題はありません。

使い方

メニューバー（Mac）/ タスクトレイ（Windows）のマイクアイコンから設定を開けます。

既定のホットキーを押しながら（または押して）話すと、文字起こしされたテキストがカーソル位置に入力されます。

プライバシーについて

録音された音声は文字起こしのため外部の音声認識サービス（OpenAI / Groq / ElevenLabs / Deepgram のいずれか、設定による）に送信されます。機密情報・個人情報の音声入力にはご注意ください。

フィードバック
