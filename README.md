# voicekey ベータ版 ダウンロード

音声入力アプリ voicekey の無料ベータ版の配布リポジトリです。
ベータテストへのご協力ありがとうございます。

**ダウンロードページ: https://tomato-1101.github.io/voicekey-releases/**
（テスターの方は上のページからどうぞ。以下は同じ内容のテキスト版です）

## ダウンロード

最新版は [Releases ページ](https://github.com/Tomato-1101/voicekey-releases/releases/latest) から:

- **Mac**: `voicekey-X.Y.Z.dmg`
- **Windows**: `voicekey-setup-X.Y.Z.exe`

API キーなどの設定は不要です。インストールしてすぐ使えます。

## Mac のインストール手順

1. ダウンロードした DMG を開き、`voicekey.app` を「Applications」フォルダにドラッグします。
2. 初回起動時に「開発元を検証できません」等の警告が出る場合は、以下の手順で開いてください。

**macOS 14 (Sonoma) の場合**

アプリケーションフォルダで voicekey.app を右クリック →「開く」→「開く」

**macOS 15 (Sequoia) 以降の場合**

一度ダブルクリックして警告を閉じる
→ システム設定 →「プライバシーとセキュリティ」→ 下の方にある「このまま開く」

※ この警告は配布方式によるもので、アプリ自体の安全性に問題はありません。
初回に一度開けば、以後は通常どおりダブルクリックで起動できます。

3. 初回起動時にマイク・入力監視・アクセシビリティの許可を求められます。
   すべて許可してください（音声入力と自動貼り付けに必要です）。

## Windows のインストール手順

1. ダウンロードした `voicekey-setup-X.Y.Z.exe` をダブルクリックします。
2. 「WindowsによってPCが保護されました」（SmartScreen）が出る場合は、
   **「詳細情報」→「実行」** の順にクリックしてください。
   ※ この警告も配布方式によるもので、アプリの安全性に問題はありません。
3. インストーラの指示に従って進めてください（管理者権限は不要です）。

## 使い方

- メニューバー（Mac）/ タスクトレイ（Windows）のマイクアイコンから設定を開けます。
- 既定のホットキーを押しながら（または押して）話すと、文字起こしされた
  テキストがカーソル位置に入力されます。

## アップデート

新しいバージョンは自動的に通知されます（毎日 1 回チェック）。
手動で確認する場合はアイコンのメニューから「アップデートを確認」を選んでください。
バグ修正・改善は自動アップデートで順次配布されます。

## プライバシーについて

- 録音された音声は文字起こしのため外部の音声認識サービス
  （OpenAI / Groq / ElevenLabs / Deepgram のいずれか、設定による）に送信されます。
- 機密情報・個人情報の音声入力にはご注意ください。
